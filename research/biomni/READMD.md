## 说明

本项目是对 [biomni.phylo.bio](https://biomni.phylo.bio)
的逆向分析. 完整的得到了:

- 文件系统设计和系统提示词

- biomni python库 (被 Biomni Agent 通过 Python 调用), 并反编译还原为 `py`
- Skills 知识库
